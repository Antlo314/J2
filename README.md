# J2 Industries — Cinematic AI Films

Premium hub site for [J2 Industries](https://www.youtube.com/@j2-industries), an AI film studio creating cinematic trailers, fan films and short scenes with Seedance 2.0, ElevenLabs and Kinovi.ai.

**2.9M+ views · 69.3K subscribers · every channel, one hub.**

## Stack

Pure static site — no build step. `index.html` + `styles.css` + `app.js`, all film thumbnails and channel art bundled in `assets/`.

## Run locally

Any static server works, e.g.:

```
python -m http.server 5602
```

then open http://localhost:5602

## Deploy

Drop the repo on any static host (GitHub Pages, Netlify, Vercel, Cloudflare Pages). For GitHub Pages: Settings → Pages → deploy from `main` branch, root folder.
